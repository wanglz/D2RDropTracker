# D2R 掉落统计器

Windows 桌面工具，用于记录《暗黑破坏神 2：重制版》的刷图次数、耗时和装备掉落。

## 当前功能

- 自动计时，按 `F8` 完成本轮并开始下一轮
- 按 `F9` 呼出窗口并聚焦掉落输入框
- 记录角色、难度、刷图区域、物品分类和品质
- SQLite 本地持久化，意外关闭后恢复未结束场次
- 汇总完成场次、掉落数、平均用时和每 100 场掉落数
- 删除错误记录并导出 UTF-8 CSV
- 置顶监控小窗，自动贴靠《暗黑破坏神 2：重制版》窗口边缘
- 小窗显示当前区域、完成次数、本轮时间和最近 5 件掉落
- 小窗不抢焦点并穿透鼠标操作，不影响游戏控制
- 小窗支持锁定/解锁：解锁时可拖动，锁定后鼠标穿透
- 完成场次和记录掉落快捷键可在 `F1` 至 `F12` 之间配置
- 关闭主窗口后缩至系统托盘，托盘菜单可恢复窗口、切换小窗或退出
- 支持撤销最近一次完成场次；当前新场已有掉落时会阻止撤销
- 每轮明确保存开始时间、结束时间和排除暂停后的有效耗时
- 场次历史支持按日期、角色、区域、难度筛选，并显示平均与最快耗时
- 支持删除错误历史场次及导出场次 CSV
- 掉落 CSV 同时包含所属场次的开始和结束时间
- 防止同时启动多个实例，避免快捷键和数据库写入冲突
- 每天首次启动自动备份数据库
- 主界面提供备份管理：立即备份、查看备份目录、校验并恢复备份
- 恢复前自动创建当前数据库的安全副本
- 最近掉落支持双击或按钮编辑名称、分类和品质
- 场次筛选同步计算掉落数量、物品统计和每 100 场掉率
- 未处理异常自动记录到本地日志
- 删除掉落后可在当前程序会话中一键撤销
- 监控小窗支持调整透明度、整体缩放和最近掉落显示条数
- 快捷键支持 `Ctrl`、`Alt`、`Shift` 组合，并可单独设置显示/隐藏小窗和保存截图
- 物品名称支持搜索补全，选择词库条目后自动推断分类和品质
- 偏好设置中可检查数据库完整性并设置备份保留天数
- 内置物品词库改为 JSON，并支持用户自定义物品与多个别名
- 掉落回收站持久保存，重启后仍可恢复或永久删除
- 场次历史支持编辑角色、区域、难度以及开始/结束时间
- 编辑掉落时可将物品移动到其他场次
- 可生成携带 .NET 运行时的 Windows x64 独立压缩包
- 场次支持记录玩家人数、MF、标签和备注，并可按标签筛选
- 新增近 14 天场次、区域平均耗时和掉落分类原生图表
- 支持一键截取主屏幕，截图保存在本地待人工识别
- 提供无第三方测试框架的自动化测试，覆盖迁移、回收站、编辑和统计

数据默认保存在：

```text
%LOCALAPPDATA%\D2RDropTracker\tracker.db
```

每日备份保存在：

```text
%LOCALAPPDATA%\D2RDropTracker\Backups
```

异常日志保存在：

```text
%LOCALAPPDATA%\D2RDropTracker\Logs
```

截图保存在软件所在目录：

```text
统计数据\截图
```

运行自动化测试：

```powershell
.\test.ps1
```

## 轻量发布

双击 `发布独立版.bat`，默认生成轻量版：

```text
publish\D2RDropTracker-lite.zip
```

轻量版不打包完整 .NET 桌面运行时，因此体积很小。运行环境需要安装
.NET 8 Desktop Runtime；Windows 已安装时可直接运行 `D2RDropTracker.exe`。

## 开发运行

```powershell
$env:DOTNET_CLI_HOME="$PWD\.dotnet-home"
.\.dotnet\dotnet.exe run --project .\D2RDropTracker
```

## 直接使用

项目已生成 Release 版本，双击根目录中的：

```text
启动D2R掉落统计器.bat
```

修改源码后，可右键使用 PowerShell 运行 `build.ps1` 重新构建。
